genmod - Make Submodule

Command Line

genmod Options [-m] Name Number
Options
Standard options, see Standard Command Line Options
-m
Make mountain (works after pwkond).
Name
Module name.
Number
Submodule number.

Input Files

Name.sub
Submodule information.
Name.v
Mountains.

Output Files

Name.sNumber, Name.mNumber
The result (see decription).

Description

This program makes a submodule, i.e., it takes the mountains contained in that submodule and spins them up. The result is written to Name.sNumber. The program assumes that the submodule structure has been calculated with mksub.

If the -m option is used, Number is treated as the number of a mountain. This mountain is generated and written to Name.mNumber.
