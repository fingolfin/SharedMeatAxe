zro - Random Orders

Command Line

zro [Options] [-Gs] Count Gen [Gen ...]
Options
Standard options, see Standard Command Line Options.
-G
GAP output.
-s
Summary output.
Count
Number or orders to calculate.
Gen
Generator.

Input Files

Gen
Generator (matrix or permutation).

Description

zro calculates the order of Count random elements of the group generated by a set of matrices or permutations. This information can be helpful to find out which group is generated by a given set of matrices or permutations.
