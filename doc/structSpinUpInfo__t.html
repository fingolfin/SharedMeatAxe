Detailed Description

Spin-up Parameters.

The SpinUpInfo_t data structure is used to pass additional parameters to the spin-up algorithm, and to return extended results to the caller. Result is set by SpinUp() to report the success of the spin-up. Possible values are 0 (successful), 1 (not found), and -1 (error).
